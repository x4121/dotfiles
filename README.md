# dotfiles
My dotfiles

## Contents
* terminal colors
* zsh config
* vim config
* tmux config

## Install
1. `git clone https://github.com/x4121/dotfiles ~/.dotfiles`
1. `cd ~/.dotfiles`
1. `./install.sh`
1. `vim +PluginInstall +qall`
